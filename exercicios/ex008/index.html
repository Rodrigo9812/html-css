Palavras e frases em Negrito

Agora com o html5 podemos colocar as letras e textos em negrito usando a tag STRONG ,para dar força a palavra ou frase digitada.

Tambem temos o Italico

Alem do negrito, para dar enfase em alguma frase ou palavra usamos a tag "EM" para dar Enfase a frase.

Juntando as tags

Tambem é possivel juntar as tags de negrito e italico

Textos Marcados

Para deixar um texto marcado usamos a tag "MARK", ele vai ser nossa caneta marca textos.
Alem disso podemos mudar a cor da marcação usando o parametro "style" e alterando a cor, porem a tag é individual, existe um meio de adicionar a mesma cor a todas as tags mark em CSS adicionando no "head" a tag "style" e selecionando a tag "mark", em seguida escolhendo a cor para as tags, para funcionar as tags mark nao podem ter cores nenhuma, deve estar apenas "mark".

Tamanho das letras

Antigamente em html5 era possivel aumentar a letra usando a tag BIG, porem essa tag ficou obsoleta, hoje em dia essa alteraçao é feita em CSS,
contudo, ainda é possivel diminuir as palavras ou frases usando a tag small.

Textos deletados

Tambem é possivel adicionar Textos deletados a seus paragrafos, basta utilizar a tag "del", para colocar uma linha em textos que nao sao mais relevantes ao paragrafo e que nao se deve dar mais tanta importancia.

Textos inseridos

O contrario dos textos deletados, temos os textos inseridos que servem para indicar um ponto muito importante e que se deve ter mais atençao a ele, para isso usamos a tag "ins".

Textos sobescritos e subscritos

Temos tambem a tag que vai nos ajudar a colocar textos como H2O e X 2+23 aos nossos textos, as tags "sub" e "sup" colocaram os elementos em cima ou embaixo do texto.

Codigos Fonte e Pre-Formatado

Temos tambem tags para colocar codigos dentro dos nossos paragrafos, como por exemplo:

num=int (input ("digite um numero")) if num % 2 == 0 print(if 'o numero {num} é par') else: print (if 'o numero {num} é impar') print ('Fim do pragrama')

esse é um codigo em Python, a tag "code" permite que coloquemos codigos como esse dentro dos nossos textos, porem perceba que não fica facil de visulizar, nao esta no formato que o codigo foi escrito. Usando apenas a tag "code" o codigo nao fica monoespaçado, logo, cada letra tem um tamanho diferente, Para ajustar usamos a tag "pre" dentro da tag "code",para deixar o codigo do jeito que ele foi escrito e para ficar monoespaçado. 

    num=int (input ("digite um numero"))
    if num % 2 == 0
    print(if 'o numero {num} é par')
    else:
    print (if 'o numero {num} é impar')
    print ('Fim do pragrama')

Perceba que agora o codigo esta mais facil de visualizar, para alterar a posiçao ou espaçamento basta selecionar o codigo e apertar "Alt" e "Tab" para colocar mais pro canto, para colocar ligeiramente mais pro meio basta selecionar o codigo e apertar apenas "Tab"

Citaçoes

Para adicinar citaçoes colocamos a tag "q", ex:
Lembre-se de que você mesmo é o melhor secretário de sua tarefa

Voce pode tambem adicionar frases maiores e de terceiros e fazer com que o navegador saiba que aquela citaçao é de um determinado livro, ebook ou jornal, basta usar a tag "blockquote", a tag sozinha ira deslocar a citação indicando que ele é de terceiros, para os mecanismos de busca entenderem de quem é a citaçao temos que adicinar o parametro "cite" junto da URL da citaçao, ex:

Lembre-se de que você mesmo é o melhor secretário de sua tarefa, o mais eficiente propagandista de seus ideais, a mais clara demonstração de seus princípios, o mais alto padrão do ensino superior que seu espírito abraça e a mensagem viva das elevadas noções que você transmite aos outros.

Abreviaçoes

Em Html5 podemos fazer abreviaçoes para certos tipos de palavras como o proprio HTML fazendo com que, quando passar o mouse em cima da abreviaçao o seu significado aparecera, um outro exemplo de abreviaçao... CSS..

Palavras ao contrario
