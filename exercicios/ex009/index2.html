Listas em HTML 🗒

Temos as Ordered Lists

Em Html5 conseguimos fazer alguns tipos de listas uma delas é a ordered list, que serve para listas como receita de bolos, lista de aprovados no vestibular,
sao listas onde a ordem dos itens importa..

  1. Item
  2. Item
  3. Item
  4. Item
  5. Item

Podemos gerar outros tipos de Ordered lists

Existe tambem listas com o marcador type da tag ol

  1. item 1
  2. Item 2
  3. Item 3

  3. Item 3

Exemplo de lista alfabetica minuscula

  3. Item 3

Exemplo de listas com algarismos romanos maiusculo

Exemplo de listas com algarismos romanos minusculo

Tambem temos as Unordered Lists

Com as unordered list, conseguimos fazer listas onde a ordem dos itens ao importa e usar simbolos diferentes como:

  • Item 1
  • Item 2
  • Item 3

Por padrão a tag ul vem com a bolinha preta preenchida mas tambem podemos usar simbolos como:

  • Item 1
  • Item 2
  • Item 1
  • Item 2
  • Item 1
  • Item 2

Misturando as listas

Poemos tambem misturar as listas tanto ordenadas como desordenadas, como por exemplo:

  1. Item 1
  2. Item 2
    • Item 2.1
    • Item 2.2
  3. Item 3
  4. Item 4

Listas de Definição

Tambem podemos criar listas de definiçoes para sites, listas como um dicionario, nao sao listas ordenadas,
dessa vez temos termos e suas descrições, por exemplo:

HTML
Linguagem de marcação para criar conteudo a sites
CSS
Linguagem de marcação para especificação de estilos em sites
JavaScrip
Linguagem de programação para criar interação aos sites
